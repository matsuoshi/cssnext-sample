# cssnext-setup

cssnext 環境のサンプルです

for CSS Talk Vol.2 (2017.01.28)  
https://taminc.doorkeeper.jp/events/54947

## つかいかた

1. `npm install` で PostCSS / cssnext 関連のパッケージをインストール
- `.postcssrc.json` ファイルを編集し、読込み元や出力先フォルダを指定
- `npm start` で watch & build 開始
